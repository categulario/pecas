Scripts de JavaScript para EPUB

Estos scripts implementan algunas funcionalidades adicionales a los EPUB.

Implementación

Solo vincula el archivo .js adentro de la etiqueta head de los XHTML del EPUB, por ejemplo:

                <script type="text/javascript" src="script.js"></script>

Documentación

Para saber cómo emplear cada uno de los scripts solo es necesario leer las líneas comentadas al inicio de estos archivos.

Scripts

  • poetry.js. Posibilita un control ortotipográfico cuando el verso excede el tamaño de la caja.

CUIDADO. Este script tiene errores que aún no han sido arreglados.

  • zoom.js. Posibilita el aumento o disminución del tamaño de la tipografía en EPUB de diseño fijo.
